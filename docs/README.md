# Документация проекта

## Общее описание решения
Программы в данном проекте помогают решать геометрические задачи.
Структура проекта:
circle.py - функции для работы с кругом
square.py - функции для работы с квадратом
utest

## Описание каждой функции с примерами вызова

### Модуль circle.py

#### Функция area(r)
Вычисляет площадь круга по заданному радиусу.
Формула: S = π × r²
Параметры:
r: радиус круга
Возвращаемое значение: площадь круга
Пример вызова:
r = 5
Вывод: 78,53981633974483

#### Функция perimeter(r)
Вычисляет длину окружности по зрааданному радиусу.
Формула: L = 2 * pi * r
Параметры: r - радиус круга
Возвращаемое значение: Длина окружности
Пример вызова:
r = 10
Вывод: 62,83185

### Модуль square.py

#### Функция area(a):
Вычисляет площадь квадрата по длине стороны.
Формула: S = a*a
Параметры:
a: длина стороны
Возвращаемое значение: площадь квадрата
Пример вызова:
a = 56
Вывод: 3136

#### Функция perimeter(a):
Вычисляет периметр квадрата по длине стороны.
Формула: S = 4 * a
Параметры: 
a: длина стороны
Возвращаемое значение: периметр квадрата
Пример вызова:
a = 5
Вывод: 20
